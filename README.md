Die Übungsleitstellensoftware dient zur Simulation einer Einsatzzentrale.

Abgebildet werden die Funktionen, die für die Erstellung, Bearbeitung und Dokumentation von Einsätzen, sowie für die Alarmierung der eingesetzten Einheiten notwendig sind.

Es besteht kein Anspruch auf Vollständigkeit.

Die Software ist ausdrücklich nicht für den realen Einsatz im Bereich einer Leitstelle gedacht.

Der vorgesehene Anwendungsfall sind vorwiegend 24 Stunden Übungen (auch als „Berufsfeuerwehrtage“ bezeichnet), die im Rahmen der Ausbildung in der Jugendfeuerwehr stattfinden.
